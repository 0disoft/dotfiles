# 🚀 Dev-Up: 통합 개발 환경 원클릭 업데이트

**Dev-Up**은 윈도우(Git Bash) 환경의 인디 개발자를 위한 **올인원 업데이트 자동화 스크립트**입니다.
복잡한 업데이트 명령어들을 일일이 칠 필요 없이, `dev-up` 명령어 하나로 런타임, 패키지 매니저, 시스템 도구를 최신 상태로 유지합니다.

## ✨ 주요 기능

시스템에 설치된 도구를 자동으로 감지(`_has`)하여 설치된 항목만 업데이트를 진행합니다.

- **Bun**: 런타임(`bun upgrade`) 및 글로벌 패키지(`bun update -g`) 동시 업데이트
- **Node.js**: Corepack 및 pnpm 글로벌 패키지 업데이트
- **Rust**: `rustup` 툴체인 업데이트
- **Python**: `pip` 및 `uv` 패키지 매니저 업데이트
- **Deno, Flutter, Julia**: 각 런타임 및 SDK 업데이트
- **Windows 시스템**: Winget(GitHub CLI, Starship) 및 Chocolatey 패키지 업데이트
- **결과 리포트**: 작업별 소요 시간 및 성공/실패 여부 요약 출력

## 🛠 설치 및 적용 방법 (Git Bash)

이 스크립트는 **Git Bash**의 설정 파일인 `.bashrc`에 함수 형태로 등록하여 사용합니다.

### 1\. 설정 파일 열기

터미널(Git Bash)을 열고 아래 명령어를 입력하여 편집기를 실행합니다.

```bash
nano ~/.bashrc
```

### 2\. 스크립트 등록

파일의 맨 아래쪽에 제공된 `dev-up` 함수 전체 코드를 복사하여 붙여넣습니다.
*(Starship 설정이 있다면 그 아래에 배치하는 것을 권장합니다.)*

> **참고:** Bun 업데이트 로직이 포함된 최신 코드를 사용하세요.

### 3\. 변경 사항 저장

- `Ctrl + O` (저장) → `Enter` (확인)
- `Ctrl + X` (나가기)

### 4\. 설정 적용 (Reload)

터미널을 껐다 켜거나, 아래 명령어를 입력하여 설정을 즉시 적용합니다.

```bash
source ~/.bashrc
```

## 💻 사용 방법

터미널에서 언제든지 아래 명령어를 입력하세요.

```bash
dev-up
```

### 실행 예시

```text
==> Bun 업그레이드
  ✓ Bun 런타임 업그레이드
  ✓ Bun 글로벌 패키지 업데이트

==> pnpm 글로벌 패키지 업데이트
  ✓ pnpm 글로벌 패키지 업데이트

... (중략) ...

⏱️ 작업별 소요 시간 요약
  ✓ Bun 런타임 업그레이드: 2s
  ✓ Bun 글로벌 패키지 업데이트: 1s
  ... Bun: SKIPPED
  ✓ pnpm 글로벌 패키지 업데이트: 5s

✅ 모든 작업 완료! (총 소요 시간: 12초)
```

## ⚠️ 주의 사항

- **관리자 권한:** Winget이나 Chocolatey 업데이트는 관리자 권한이 필요할 수 있습니다. 권한 오류가 발생하면 Git Bash를 \*\*"관리자 권한으로 실행"\*\*해 주세요.
- **pnpm 경고:** "Ignored build scripts" 경고가 발생하면 `pnpm approve-builds -g`를 실행하라는 안내 메시지가 출력됩니다.
